# 怎么食用 How to use？

运行**hitokoto.php**直接返回字符串。如 Run **hitokoto.php** can return sting.Such as
>*我想找旧版本金蛋服务器 --Indev_Classic*

附加参数如**hitokoto.php?json**可以返回json。如 Additional parameters such as **hitokoto.php?json** can return json. Such as

`{"hitokoto":"也不想再偷别人东西了","from_who":"Indev_Classic","from":"Indev_Classic名言警句"}`

# 自定义 Customize？
修改**hitokoto.dat**即可更改一言句子库 Modifying **hitokoto.dat** can change the sentence database of one sentence
